Ключевые особенности

Фоновая работа

Приложение продолжает работать в фоновом режиме, обеспечивая постоянный мониторинг ваших устройств.

Группировка виджетов

Удобная организация виджетов по группам для лучшей визуализации и управления.

Сцены

Создавайте сцены для одновременной отправки сообщений на разные устройства.

Множество брокеров

Поддержка одновременной работы с несколькими MQTT брокерами.

Резервное копирование

Создание локальных резервных копий данных для защиты ваших настроек.

JSONPath

Поддержка JSONPath для работы со сложными структурами данных.
