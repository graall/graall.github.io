MQTT Dashboard Client для Android

О приложении

MQTT Client (MQTT Dashboard Client) — это мощное приложение для Android, предназначенное для работы с MQTT-протоколом. Приложение позволяет подключаться к MQTT-брокерам, подписываться на топики, публиковать сообщения и визуализировать данные в удобном dashboard-интерфейсе.

Премиум-функции

  • XML, JSON, Regex парсинг данных
  • Regex-парсинг с извлечением временной метки
  • Префикс и постфикс для входящих данных
  • Множественные скрипты с условиями И/ИЛИ
  • Сценарий оповещения звуком
  • Отправка заданного топика при запуске программы
  • LogicalBlock с группировкой условий
  • Загрузка сторонних SSL-сертификатов (PEM)
  • Самоподписанные сертификаты
  • XY формат для Color widget В разработке

Бесплатные функции

  • Префикс и постфикс для исходящих данных
  • JsonPath парсинг данных и времени
  • Гибкая настройка виджетов на главном экране
