About the Application

MQTT Client (MQTT Dashboard Client) is a powerful Android application designed to work with the MQTT protocol. The application allows connecting to MQTT brokers, subscribing to topics, publishing messages, and visualizing data in a convenient dashboard interface.

Premium Features

  • XML, JSON, Regex data parsing
  • Regex parsing with timestamp extraction
  • Prefix and postfix for incoming data
  • Multiple scripts with AND/OR conditions
  • Sound alert scenario
  • Sending a topic on app startup
  • LogicalBlock with grouped conditions
  • Third-party SSL certificates (PEM)
  • Self-signed certificates
  • XY format for Color widget In Development
