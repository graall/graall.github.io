Переход на российский магазин приложений для пользователей из РФ

Для пользователей из РФ: В связи с ограничениями Google Play, рекомендуем перейти на версию из RuStore для возможности оплаты встроенных покупок.

Пошаговая инструкция

1Установка RuStore

  1. Перейдите с мобильного устройства по ссылке на приложение в RuStore
  2. Нажмите кнопку "Установить с помощью RuStore"
  3. Следуйте официальной инструкции по установке магазина

2Установка MQTT Dashboard Client

  1. Откройте RuStore и авторизуйтесь (необходимо для покупок)
  2. Найдите "MQTT Dashboard Client" через поиск или перейдите по прямой ссылке
  3. Установите приложение (старая версия останется на устройстве)
Обратите внимание: После установки у вас будет две версии приложения - из Google Play и из RuStore.

3Перенос данных

Для переноса данных из старой версии в новую:

Экспорт из старой версии
  1. Откройте версию из Google Play
  2. Перейдите в Настройки → Резервное копирование
  3. Создайте резервную копию
  4. Выберите папку для сохранения
Импорт в новую версию
  1. Откройте версию из RuStore
  2. Перейдите в Настройки → Резервное копирование
  3. Восстановите из резервной копии
  4. Выберите файл копии
После успешного переноса данных вы можете удалить старую версию из Google Play.
