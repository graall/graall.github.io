Политика конфиденциальности

Дата обновления: 26.02.2026 | Версия документа: 1.0

1. Общие положения

Настоящая Политика конфиденциальности разработана в соответствии с Федеральным законом Российской Федерации от 27 июля 2006 года № 152-ФЗ «О персональных данных» и иными нормативно-правовыми актами Российской Федерации в области персональных данных и применима к отношениям, возникающим при использовании мобильного приложения «MQTT Client (MQTT Dashboard Client)» (далее — «Приложение»).

Настоящая Политика конфиденциальности применяется исключительно к Приложению и не распространяется на другие приложения, сайты, продукты или услуги третьих лиц, даже если они доступны по ссылкам из Приложения.

2. Субъект персональных данных

Субъектом персональных данных является физическое лицо, использующее Приложение (далее — «Пользователь»).

3. Персональные данные, собираемые и обрабатываемые

3.1. Персональные данные, предоставляемые Пользователем:

  • Адрес электронной почты (при обращении в службу поддержки)
  • Имя пользователя (при обращении в службу поддержки)

3.2. Персональные данные, собираемые автоматически:

  • Уникальные идентификаторы устройства
  • Тип устройства и его характеристики
  • Версия операционной системы
  • IP-адрес
  • Информация о сессиях использования Приложения
  • Данные о совершенных покупках (идентификаторы транзакций)

3.3. Данные, собранные с помощью сервисов аналитики:

  • Статистика использования Приложения
  • Частота использования функций Приложения
  • Информация о сбоях в работе Приложения
  • Данные о геолокации (в обезличенном виде)

4. Цели сбора и обработки персональных данных

Персональные данные обрабатываются в следующих целях:

  • Предоставление и улучшение функциональности Приложения
  • Обработка и выполнение обязательств по договору (публичной оферте)
  • Обеспечение технической поддержки Пользователей
  • Проведение статистических исследований
  • Предотвращение нарушений условий использования Приложения
  • Направление информационных сообщений (с согласия Пользователя)

5. Правовое основание обработки персональных данных

Правовым основанием для обработки персональных данных являются:

  • Согласие Пользователя на обработку его персональных данных (п. 1 ст. 6 Федерального закона № 152-ФЗ)
  • Необходимость исполнения договора, стороной которого либо выгодоприобретателем или поручителем по которому является Пользователь (п. 3 ст. 6 Федерального закона № 152-ФЗ)

6. Передача персональных данных третьим лицам

Персональные данные могут передаваться следующим третьим лицам:

  • RuStore (АО «Сбербанк») — для обработки платежей по подписке
  • Firebase Analytics (Google LLC) — для анализа использования Приложения, сбора статистики и отслеживания сбоев
  • Yandex AppMetrica (ООО «Яндекс») — для анализа использования Приложения и маркетинговой аналитики

При передаче персональных данных третьим лицам заключены соответствующие договоры, обеспечивающие защиту прав субъектов персональных данных.

Важно: Firebase и Yandex AppMetrica могут передавать данные на серверы за пределами Российской Федерации. Используя Приложение, Пользователь даёт согласие на такую передачу.

7. Сроки обработки и хранения персональных данных

Персональные данные обрабатываются и хранятся в течение всего срока использования Приложения Пользователем, а также в течение срока, необходимого для выполнения требований законодательства Российской Федерации.

По истечении срока хранения персональные данные подлежат уничтожению или обезличиванию.

8. Права субъекта персональных данных

Пользователь имеет право:

  • Получать информацию, касающуюся обработки его персональных данных
  • Требовать уточнения, блокирования или уничтожения неполных, неточных или устаревших персональных данных
  • Отозвать свое согласие на обработку персональных данных
  • Ознакомиться с персональными данными, обрабатываемыми Оператором
  • Требовать прекращения обработки персональных данных
  • Обжаловать действия Оператора в уполномоченный орган по защите прав субъектов персональных данных

9. Обязательства Оператора

Оператор обязан:

  • Обрабатывать персональные данные в соответствии с законодательством Российской Федерации
  • Принимать необходимые правовые, организационные и технические меры для защиты персональных данных
  • При сборе персональных данных предоставить информацию об Операторе, целях и способах обработки персональных данных
  • Обеспечить конфиденциальность персональных данных
  • Осуществлять обработку персональных данных исключительно в целях, указанных в настоящей Политике
